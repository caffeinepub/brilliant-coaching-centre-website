# Specification

## Summary
**Goal:** Increase backend storage capacity to support image uploads and file storage.

**Planned changes:**
- Increase stable_memory_reserve_mb from 0 to 512 MB in backend canister settings
- Enable reliable storage for teacher profile images, student data, review photos, and document uploads

**User-visible outcome:** Teachers and administrators can upload profile images and files without storage errors, supporting smooth content management operations.
